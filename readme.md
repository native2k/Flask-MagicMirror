**Flask MagicMirror**

A simple magic mirror application built in Flask for my fiancee and I. 
The HTML isn't clean so I may add bootstrap later to help with that. 

1. Current time
    * Day of Week
    * Date
    * Month
    * Time of Day
2. Current weather
    * Temperature
        * Max
        * Min
        * Current
    * Humidity
    * Icon
        * Snow
        * Rain
        * Cloudy
        * Clear (Day)
        * Clear (Night)
3. Calendar *coming*
4. Traffic *coming*
5. Other ideas:
    * Fitbit
    * Audio 
    * Compliments

* pyowm
